# 社内Sリーグ レギュレーション

Mリーグ公式ルールに準拠した社内リーグの競技規定。

## 競技形式

- 4人打ち・半荘戦(東南戦)
- 持ち点 **25,000点**、**30,000点返し**(オカはトップに +20pt)
- 順位点(ウマ):

  | 順位 | 順位点 | オカ込みの実質 |
  |---|---|---|
  | 1位 | +30pt | **+50pt** |
  | 2位 | +10pt | +10pt |
  | 3位 | ▲10pt | ▲10pt |
  | 4位 | ▲30pt | ▲30pt |

- スコア計算式: `(最終持ち点 − 30,000) ÷ 1,000 + 順位点`(小数第1位まで)
- 同点の場合は起家に近い方を上位とする

## 主なルール

| 項目 | 採用 |
|---|---|
| 赤ドラ | あり(赤五萬・赤五筒・赤五索 各1枚、計3枚) |
| 一発・裏ドラ・槓ドラ・槓裏 | すべてあり |
| 喰いタン・後付け | あり(アリアリ) |
| 切り上げ満貫 | あり(30符4翻・60符3翻は満貫) |
| 途中流局 | なし(九種九牌・四風連打等でも続行) |
| ダブロン・トリプルロン | **なし(頭ハネ)**。放銃者から反時計回りに最も近い1人のみアガリ |
| 責任払い(パオ) | あり(大三元・大四喜・四槓子を確定させる副露をさせた場合。ツモは全額、放銃時は折半) |
| トビ | **なし**(持ち点がマイナスになっても南4局まで続行) |
| オーラスの親のアガリやめ・テンパイやめ | **なし**(アガリ・テンパイなら連荘) |
| 延長戦(西入) | なし(南4局終了時にトップが30,000点未満でも終了) |
| 流し満貫 | あり(流局時にアガリ役扱い。自分の鳴き・鳴かれ・リーチで不成立) |
| 数え役満 | なし(13翻以上は三倍満) |

> 打牌の持ち時間(考慮時間)ルールは放送用のため社内では採用しない。
> 細部で疑義が出た場合はMリーグ公式ルールに従い、都度このドキュメントに追記する。

## 反則と罰則(Mリーグ準拠)

### チョンボ(重い反則)

**罰則: 本人のトータルポイントから −20pt**(親・子問わず一律。他家への支払いはなし)。
その局は**ノーゲーム** — 同じ親で打ち直し、本場は増やさず、供託リーチ棒は返却。

| チョンボになる行為 |
|---|
| 誤ロン・誤ツモで手牌を倒した(倒牌した)場合 |
| ノーテンリーチ(発覚時、または流局時の手牌公開で発覚) |
| リーチ後の不正なカン(待ちや手の構成が変わるカン) |

※故意と認められた場合は罰則3倍(−60pt)。Mリーグでは審判が裁定するが、社内では4人協議で決める。

### アガリ放棄(軽い反則)

**罰則: その局はアガれない。**以降その局は鳴き・カン・リーチも不可。流局時は**常にノーテン扱い**(ノーテン罰符を払う)。

| アガリ放棄になる行為 |
|---|
| ロン・ツモの発声のみで倒牌しなかった場合(誤発声) |
| 多牌(14枚以上)・少牌(12枚以下) |
| 先ヅモ |
| 喰い替え |
| 空行為(空ポン・空チーなど、発声だけして行為をしない) |

### 軽微なもの

発声忘れ・打牌の置き間違いなどは、その場で注意して続行。判断に迷ったら4人で協議し、決着しなければ「軽い方の罰則」を採用する。

## 個人タイトル(Mリーグ4部門準拠)

| タイトル | 決め方 |
|---|---|
| 個人スコア賞 | シーズン累計ptの1位(MVP) |
| 最高スコア賞 | 1半荘の最高持ち点 |
| 最多トップ賞 | 1着(トップ)の回数 |
| 4着回避率賞 | 4着を引かない率 |

役満はタイトルとは別枠で、アプリの「役満バッジ」に達成者・半荘とともに永久記録される。

## シーズンと景品

- シーズンは**全65半荘(6サイクル完走)**を基本とする。変更する場合も下表の「完璧試合数」で区切る(風・並びまで完全均等)。最低でも5の倍数で区切れば全員の試合数は揃う

| 試合数 | 完走サイクル | 1人あたり | 週2ペース |
|---|---|---|---|
| 65半荘 | 6 | 52戦 | 33週 |
| 75半荘 | 7 | 60戦 | 38週 |
| 85半荘 | 8 | 68戦 | 43週 |
| 95半荘 | 9 | 76戦 | 48週 |
| 105半荘 | 10 | 84戦 | 53週 |
| 115半荘 | 11 | 92戦 | 58週 |
| 125半荘 | 12 | 100戦 | 63週(全120卓制覇) |

- シーズン試合数の切り替えはアプリの「シーズン設定」の表からワンタップ(記録係のみ)
- 座席表は第125半荘まで用意済み(第6半荘以降、全120通りの卓の組み合わせを1回ずつ使う完全設計。同じ座席の配置は二度と現れない)
- 順位ごとの景品はアプリのレギュレーション欄の最後に表示する(内容は随時グレードアップ)
- 個人タイトル4部門にはそれぞれ**1万円相当**の景品(1人で複数タイトルの独占もあり)

## リーグ運営

- 参加者: 5名(半荘ごとに1名が抜け番)
- 週に最低2半荘を消化する(3半荘以上打つ場合はアプリの座席表の順に前倒しで消化する)
- 順位は各半荘のスコア(pt)の累計で決定する
- 席(起家)は座席表の指定に従う(10半荘=1サイクルで全員が出場8回・抜け番2回・各家ちょうど2回ずつになる厳格設計)
